## Features

- **Romantic Slideshow**: Photos display one at a time with captions
- **Photo Grid**: View all photos in a beautiful grid layout
- **Random Display**: Photos shuffle each time for a fresh experience
- **Click to Enlarge**: Click any grid photo to see it in the slideshow
- **Auto-play**: Slideshow advances automatically every 3 seconds

## Notes

- Photos display randomly from the `pictures` folder
- The app will work with emoji placeholders if no photos are added
- For best results, use photos with similar aspect ratios
- Add as many photos as you want!
